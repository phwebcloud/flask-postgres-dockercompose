# quero-python

Dependencies:

* Python 3.7

Instale as dependências:

```
pip install -r requirements.txt
```

Configure a aplicação:

```
export DB_HOST = <host>
export DB_NAME = quero
export DB_USER = <db-user>
export DB_PASSWD = <db-passwd>
```

Para executar a aplicação localmente execute:

```
python app.py
```
